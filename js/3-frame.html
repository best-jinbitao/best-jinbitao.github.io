# 框架

# Vue

是一个用于创建用户界面的开源JavaScript框架，也是一个创建单页应用的Web应用框架。

# React

React是用于构建用户界面的JavaScript库，起源于Facebook的内部项目。

# Angular

AngularJS是一种为动态web应用设计的框架。它使用HTML作为模板语言，并且扩展了HTML的语法，让开发者能够更清晰、更简洁地开发应用组件。
